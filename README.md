3D Visualization of Letter D
Описание

Веб-приложение для построения и визуализации трёхмерного объекта — буквы D — в виде каркасной модели.
Позволяет применять трёхмерные преобразования и отображать объект в разных проекциях.

Функциональность

Построение каркасной модели буквы D.

Преобразования:

Перенос по осям X, Y, Z.

Масштабирование по осям X, Y, Z.

Вращение вокруг произвольной оси.

Визуализация:

Перспективная проекция.

Ортографические проекции: XY, XZ, YZ.

Настройки:

Отображение осей.

Регулировка толщины линий.

Вывод текущей матрицы преобразования.

Структура проекта

index.html — интерфейс приложения и канвасы.

style.css — стили для канвасов и элементов управления.

script.js — логика построения модели, преобразований и рендеринга.

Использование

Открыть index.html в браузере.

Выбрать проекцию для основного канваса.

Использовать поля и кнопки для:

Переноса (tx, ty, tz)

Масштабирования (sx, sy, sz)

Вращения (ax, ay, az, angle)

Нажать Применить для выбранного преобразования.

Нажать Сброс для возврата к исходной позиции.

Нажать Экспорт матрицы, чтобы получить текущую матрицу преобразования.
